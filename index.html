Poniżej są trzy numerowania z różnymi znakami numerowania


Wybrane operatory w języku c++:

  • a + b = dodawanie
  • a - b = odejmowanie
  • a * b = mnożenie
  • a / b = dzielenie
  • a < b = znak mniejszości
  • a <= b = mniejszy lub równy
  • a > b = znak większości
  • a > = b = większy lub równy
  • a == b = znak równości
  • a != b = różny od
  • a && b = iloczyn logiczny
  • a || b = suma logiczna
